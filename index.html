⚡ Оптимизация производительности

  • Добавлена оптимизация фарма с новыми способами определения ошибок и автоматическим решением
  • Переработано распределение ресурсов - программа потребляет на 10% меньше ресурсов

🚀 Новые настройки

  • Возможность указать ссылку на актуальный сайт HyperDrop (помогает при смене домена сайтом)
  • Добавлен БЕТА режим прокси:
    • Работает только с конфигами от @abvpnbot
    • Позволяет фармить даже при блокировке сайта в РФ
    • Автопоиск рабочего сервера и свободного порта
    • НЕ влияет на систему - страна меняется только для программы
  • Настройка фильтра серверов

⚙️ Улучшения

  • Добавил в папку программы файл bat: Менеджер автозагрузки, с его помощью можно добавить программу и отдельно сам фарм в автозагрузки windows!

🎨 Улучшенные логи

  • Обновлён визуал логов - теперь красивее и приятнее!
  • При старте фарма:
    • Количество аккаунтов
    • Количество потоков
    • Режим работы
    • Состояние прокси
    • Время запуска
  • Оповещение о партии аккаунтов (какие аккаунты сейчас фармятся)
  • Лог отдельного аккаунта:
    • Время фарма
    • Количество попыток угадывания
    • Успешно ли прокручены все 30
    • Собраны ли монеты
    • Текущий баланс
    • Хватает ли монет для кейса
    • Сколько осталось фармить до калаша
  • Промежуточная статистика:
    • Сколько обработано
    • Сколько успешно обработано
    • Сколько осталось обрабатывать
    • Время обработки текущей партии
  • Итоговая статистика:
    • Общее время фарма
    • Аккаунтов обработано
    • Успешно / Ошибки
    • Среднее время фарма 1 аккаунта
    • Общий баланс
    • Всего прокручено
    • На каких аккаунтах есть калаши
    • На каких аккаунтах нет калашей
    • Самый быстрый аккаунт
    • Самый медленный аккаунт

Внимание: Возможны баги, сообщить в лс. Возможно что-то забыл добавить в ChangeLog)

🔧 Технические детали и установка

Версия 3.48 использует следующие библиотеки:

  • customtkinter 5.2.2 - для современного пользовательского интерфейса
  • playwright 1.42.0 - для надежного взаимодействия с веб-интерфейсом
  • python-telegram-bot 20.8 - для интеграции с Telegram
  • pyperclip 1.8.2 - для работы с буфером обмена
  • keyboard 0.13.5 - для реализации горячих клавиш
  • win10toast 0.9 - для системных уведомлений
  • requests - для HTTP запросов
